# Git attribution

Commits on `briancrabtree-me/*` repos use Brian Crabtree as author. Agent-assisted work may include:

```
Co-authored-by: BlankLogic <git@blanklogic.io>
```

Cursor/cursoragent co-authors are blocked. See [server-context docs](https://github.com/briancrabtree-me/server-context/blob/main/docs/public-github-git-hygiene.md).
